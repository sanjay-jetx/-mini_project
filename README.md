# 🐍 Python Practice & Mini Projects

This repository contains Python programs and mini projects developed while learning core programming concepts and problem solving.

---

## 🚀 Projects Included

### 👨‍💼 Employee Management System

* Manage employee records (add, update, delete)
* Search by ID and name
* Salary reports (highest, lowest, average)
* Auto-save data using file handling
* Basic login system

---

### 🎮 Hangman Game

* CLI-based word guessing game
* Uses loops and conditions
* Interactive gameplay

---

### 🏷️ Auction System

* Accepts bids from multiple users
* Finds highest bidder using dictionaries

---

### 🔐 Password Masker

* Masks user password input for privacy
* Prevents password visibility during typing

---

### 🔐 Caesar Cipher

* Encrypts and decrypts messages
* Demonstrates string manipulation

---

### 🛒 E-commerce Discount System

* Calculates final price based on discount rules
* Uses conditional logic
